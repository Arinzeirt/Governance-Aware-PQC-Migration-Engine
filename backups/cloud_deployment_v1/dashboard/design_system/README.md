# Enet Design System (EDS)

The Enet Design System (EDS) provides the shared visual language used across all Enet Technologies products.

## Purpose

The design system ensures visual consistency across:

- Enet Corporate Website
- Enterprise Quantum Migration Platform (EQMP)
- Research Centre
- Developer Portal
- Future Enterprise Applications

## Modules

- colors.py
- typography.py
- spacing.py
- radius.py
- shadows.py
- css.py
- theme.py

## Principles

- Consistency
- Accessibility
- Enterprise First
- Reusability
- Scalability

All UI components should consume tokens from this package rather than defining styles locally.
